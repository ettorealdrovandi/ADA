# About ADA

**ADA — Accessible Documents from LaTeX** is a research and
experimentation playground for emitting accessible PDF and HTML output
from LaTeX (and Markdown) sources.

## Author

Ettore Aldrovandi.

## License

Released under the [MIT License](https://github.com/ettorealdrovandi/ADA/blob/main/LICENSE).

## Status

ADA is an active experiment, not a finished product. Layouts,
filenames, and build scripts may change as the underlying tagging
infrastructure evolves and as more of the directory is restructured.
Use it as a reference and a starting point, not a stable API.
