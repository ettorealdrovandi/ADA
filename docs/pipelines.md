# Pipelines

ADA compares four toolchains for emitting accessible documents from
LaTeX or Markdown sources:

- [Tagging (PDF/UA)](pipelines/tagging.html) — tagged PDF via LuaLaTeX + `tagpdf`.
- [LaTeXML](pipelines/latexml.html) — HTML5 + MathML via LaTeXML.
- [TeX4ht](pipelines/tex4ht.html) — HTML5 via `make4ht`, with MathJax and MathML variants.
- [Markdown](pipelines/markdown.html) — single-file Markdown to HTML5 via Pandoc.

See the [comparison page](comparison.html) for a one-look summary.
